Leetcode 88 Merge Sorted Array

题目来源

Merge Sorted Array
本题目算作是端午节题目，哈哈，虽然有点晚了。

分析题目

这个题目看上去是很容易理解的。
将两个有序的数组归并到同一个数组中，假定第一个数组有着足够大的空间容纳所给定的两个数组的成员。
题目的参数误导了我好一会，因为给定的是有效数据的数量，一开始被我误认为是数组的长度，因此有了给定nums1=[0],m=0,nums2=[1],n=1这样的数据的时候不能理解，明明有个0，为什么长度为0.后来看到了这个例子：
丫的这个图片怎么回事。
总之，就是说这个nums1中的0，代表没有数据，此处可以用来存储的意思，对就是这样，所以nums1=[0],m=0的意思就是，nums1中是空的，可以用来存储nums2中的一个数据。

分析结果

主要影响就是上述的问题，解决了之后就比较好办了，我们采取和之前题目差不多的计数方法（暂时叫这个吧），以一个数记录下标，将有效值放入下标指定位置，之后改变下标。
对于本题，倒序处理比较好（看了人家代码），因为是在nums1中存储最终的数据，所以可以直接从m+n-1的下标开始，并不会影响到nums1中前面的数据。
而且，在某一个数组数据处理结束之后，若为nums1结束则不需处理，只处理nums2即可。
明天写一下正序的。

结果代码

代码技巧

nums1[res--] = nums1[i--];
写法节省空间，意思也明了。
