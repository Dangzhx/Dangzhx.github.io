Leetcode 28 Implement strStr()

题目来源

Implement strStr()

发现了为什么字体变大，只要挨着 - - - 就会变成加粗模式

题目分析

还是leetcode的简单题目，我的目的是先好好做简单题目，以前虽然中等困难的也做过一些，不过不少题目由于基础不扎实或者对于题目不够了解而导致做的效果很不好，因此先做好简单题目打好基础，当然同时也要巩固学习数据结构。
近几天的做题并没有什么经典的算法，都是凭借思路解决问题的题目。好了，说正题。今天的题目，是要实现一个常用方法，String类的方法indexOf ( String anotherstring ) ，即寻找一个字符串在另一个字符串中第一次出现的位置，若是没有则返回-1，有的话就返回位置索引。

注意点

  • 给了空的输入，这是leetcode一贯的作风，两个空的输入一定会有。 这时应当返回0.
  • 上一条可以归结为，只要第二个串的长度为0，那么结果就是0.
  • 可能出现第二个串长度大于第一个的情况，要进行处理。
  • 取子字符串时后，substring方法的参数相同取到的时空值，该方法取得的值为两个参数之间的值。
  • 在正常情况下，第一个字符串中符合条件的子串一共有len1-len2个。

心得吧

一定要好好想清楚逻辑关系再进行代码编写，不然会出现碰到一个错误例子解决一个错误例子，最后导致代码又臭又长，效率底下，惨不忍睹。
要尽量简化代码，简便易懂的代码才是最好的。
