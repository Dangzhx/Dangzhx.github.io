Leetcode 14 Longest common prefix

题目来源

Longest Common Prefix

题目分析

题意很好理解，找出字符串数组中的相同的最长前缀，给定许多字符串，找到共同最长前缀，需要用到的是字符串的处理方法。

思路

拿出一个字符串之后，这个字符串可能就是最短，或者不是，那么遍历其他所有的字符串，每一次遍历中都寻找共同的前缀，如若没有，那么结果为空。

方法总结

题目较为简单，却走了很多弯路，源于对方法的不熟悉。
res.indexOf(String subString)方法，如果subString是res的字串，则返回subString在res中第一个字母的索引，如果不是，则返回-1，因此只要用返回值是为0，那么就是前缀。

另外，一般使用equals比较两个字符串，返回值为布尔值。
