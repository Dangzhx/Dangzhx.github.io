Leetcode 67 Add Binary

题目来源

Add Binary

题目分析

这道题目是让我们实现二进制的加法，并且是给定两个字符串形式的二进制数进行相加。我的思维受到了局限，忘记了把字符处理为数字处理更为方便，不过我认为还是经验问题，这次见到了，下次大概就会用到了。
还有StringBuilder这个类，平时也没怎么用过，这时候发现确实很方便，append()可以把各种类型的量添加到字符串中，不过后面要用toString()转化为字符串就是了。
思维还是一位一位的处理，每一次都要有进位，每一次结果添加到结果字符串中，如果最后还有进位，那么就再加个’1’就好了。
