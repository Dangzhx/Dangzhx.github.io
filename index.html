Linux ubuntu 环境搭建

题记
因为要用redis，redis官方不支持windows，尽管有人维护了redis在windows上，总会有点问题，所以还是好好弄个linux吧。
不记得自己之前的linux系统密码了，不知道怎么回事还进入不了grub系统，就删了吧，反正也不记得之前里面有什么了。
这里重新弄一个。
Ubuntu下的C/C++环境搭建：
ubuntu国内下载地址

  • 系统：Ubuntu 16.0.4
  • vmware虚拟机
  • 先找到开源社区下载镜像（搞清楚了i386和amd64：x86-64的区别,领先intel达到64位桌面cpu的amd）找到16.0.4 desktop amd64 iso等关键字。
  • 按照步骤安装了镜像，1Gram，20Grom.
  • gcc和vim都自带了，可能缺一点头文件影响编译。sudo apt-get install build-essential
  • 接下来还是弄个ide吧，自己水平不足，不能只用vim。就qt吧。
  • 需要root权限，这时候无法输入密码。哦，输入密码看不见不是无法输入。
  • 可以了，gcc hello.c -o hello, ./hello即可执行（把main拼写成mian看不见，会有return错误）。
  • ok，基本环境先到这里。下次是Qt creator和Redis。
